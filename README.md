# solium-plugin-dotta

> Dotta's Solium plugin

## Install

```
$ npm install -g solium
$ npm install -g solium-plugin-dotta
```

## Usage

In the .soliumrc.json file, add:

    {
      ...
      "rules": {
        ...
        "dotta/underscore-function-arguments": [
          "warning"
        ]
    }
